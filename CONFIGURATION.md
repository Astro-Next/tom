# Configuration

Tom's configuration lives in `.tom/settings.json`, created by `tom init`.

## Default settings

```json
{
  "id": "a1b2c3d4e5f6",
  "patrol": {
    "interval": "30m"
  },
  "retro": {
    "interval": "1d",
    "time": "22:00"
  },
  "agent": {
    "timeout": "30m",
    "maxRetries": 2
  },
  "dev": {
    "concurrent": 2
  },
  "review": {
    "concurrent": 2
  }
}
```

## Fields

| Field | Default | Description |
|-------|---------|-------------|
| `id` | auto | Unique project ID, generated by `tom init` |
| `patrol.interval` | `"30m"` | How often patrol runs (`30m`, `1h`) |
| `retro.interval` | `"1d"` | How often retro runs (`1d`, `2d`) |
| `retro.time` | `"22:00"` | When retro runs (24-hour `HH:MM`) |
| `agent.timeout` | `"30m"` | Max time for any agent subprocess |
| `agent.maxRetries` | `2` | Failed attempts before blocking/giving up |
| `dev.concurrent` | `2` | Max simultaneous dev agents |
| `review.concurrent` | `2` | Max simultaneous review agents |

## Interval format

- Minutes: `30m`
- Hours: `2h`
- Days: `1d`

All interval and timeout fields must match the pattern `<digits><unit>` (e.g. `30m`, `2h`, `1d`). `retro.time` must be valid 24-hour `HH:MM` (00:00–23:59). `agent.maxRetries`, `dev.concurrent`, and `review.concurrent` must be positive integers.

## Labels

Tom manages a fixed set of GitHub labels. Run `tom labels` to create or update them. Use `tom labels --clean-labels` to also delete labels on the repo that Tom does not manage. `tom init` also accepts `--clean-labels`.

## Project files

Tom expects these in the repository it manages:

| File | Purpose |
|------|---------|
| `.tom/settings.json` | Configuration |
| `CLAUDE.md` | Agent instructions |
| `CONVENTIONS.md` | Coding standards |
| `docs/index.md` | Project knowledge base |
| `.worktrees/` | Agent worktrees (gitignored) |

## Runtime files

Runtime files are stored outside the repo:

| Path | Purpose |
|------|---------|
| `~/.tom/{id}/patrol.lock` | Patrol process lock file |
| `~/.tom/{id}/retro.lock` | Retro process lock file |
| `~/.tom/{id}/logs/tom.log` | Runtime logs |
| `/tmp/tom-{id}/cache/` | Attachment cache |
